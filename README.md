# WareEPE - Warehouse Management System

A comprehensive warehouse management system designed for export processing enterprises, built with modern web technologies and deployed on the cloud.

## Overview

WareEPE is a professional warehouse management solution that helps businesses efficiently manage their inventory, track materials, and streamline operations. The system provides real-time insights into stock levels, automates inventory processes, and ensures accurate record-keeping for export processing enterprises.

## Key Features

**Inventory Management**
- Raw material inbound and outbound tracking
- Auxiliary material management
- Finished product inventory control
- Real-time stock level monitoring

**Production Management**
- Product recipe management
- Production planning and tracking
- Quality control integration
- Batch and lot tracking

**Financial Management**
- Cost calculation and analysis
- Profit margin tracking
- Financial reporting
- Budget planning tools

**User Management**
- Role-based access control
- Secure authentication system
- User activity logging
- Permission management

**Data Analytics**
- Inventory turnover analysis
- Trend reporting
- Demand forecasting
- Performance metrics

## Technology Stack

**Frontend**
- React 19 with modern hooks
- React Router for navigation

- Responsive CSS design

**Backend**
- Node.js with Express framework
- PostgreSQL database

**Deployment**

# WareEPE - 仓库管理系统

专为出口加工企业设计的综合仓库管理解决方案，采用现代网络技术构建并部署在云端。

## 系统概述

WareEPE是一个专业的仓库管理解决方案，帮助企业高效管理库存、跟踪物料并简化运营流程。系统提供实时库存洞察、自动化库存流程，并确保出口加工企业的准确记录管理。

## 核心功能

**库存管理**
- 原料入库和出库跟踪
- 辅料管理
- 成品库存控制
- 实时库存水平监控

**生产管理**
- 产品配方管理
- 生产计划和跟踪
- 质量控制集成
- 批次和批号跟踪

**财务管理**
- 成本计算和分析
- 利润率跟踪
- 财务报表
- 预算规划工具

**用户管理**
- 基于角色的访问控制
- 安全认证系统
- 用户活动日志
- 权限管理

**数据分析**

- 趋势报告
- 需求预测
- 绩效指标
